# Jupyter Data Directory

[![Join the Gitter Chat](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/QuantStack/Lobby?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)

A Jupyter server extension adding a static handler serving a given directory of the filesystem.

## Configuration

Change the served directory by editing the `etc/jupyter/jupyter-data-directory.json` configuration file.
